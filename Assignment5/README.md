# Assignment 5 - Travel Booking Website (Node.js + MongoDB)

## 📌 Problem Statement
Create a functional web application using Node.js and Express with a NoSQL database like MongoDB.

---

## 🎯 Objective
To build a dynamic travel booking website where users can:
- View travel packages
- Book trips
- Store bookings in a database
- View previous bookings

---

## 🌐 Features
- 🏠 Home Page (Hero section with modern UI)
- 📦 Packages Page (Card-based layout with images)
- 📝 Booking Page (Form submission)
- 📋 Bookings Page (Stored data from MongoDB)
- 🎨 Modern UI with animations, hover effects, and gradients

---

## 🛠️ Technologies Used
- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS (Template Engine)
- HTML, CSS (Glassmorphism UI)
- Bootstrap (optional components)

---

## 🧠 How It Works
1. User visits homepage and explores packages  
2. Selects a package and fills booking form  
3. Booking is stored in MongoDB  
4. User can view all bookings on the bookings page
